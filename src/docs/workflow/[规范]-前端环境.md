### node

nodejs官网下载[[https://nodejs.org/en/ | 安装包]] 
或者
命令行
```
# Mac下命令行安装nodejs
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
brew install node
```
测试
```bash
node -v
```

## git

### 设置换行符
```bash
# 系统设置提交时转换为LF，检出时不转换
git config --global core.autocrlf input
```
