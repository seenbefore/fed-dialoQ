## pdf在线预览

### 基础属性

| 参数             | 说明                     | 类型                                  | 可选值         | 默认值 |
| ---------------- | ------------------------ | ------------------------------------- | -------------- | ------ |
| proxyPrefix      | 代理前缀                 | string                                | —              | -      |
| dataType         | 数据类型                 | [DateType](#DateType)[]         | url,stream,bytes | url   |
| data             | pdf文件数据              | string、object                        | -                | -

### DateType

| 参数             | 说明                     | 类型
| ---------------- | ------------------------ | ------------------------------------- |
| url              | pdf文件地址             | string                               |
| stream           | 文件流对象              | object                               |
| bytes            | 文件流字节数组           | string                               |
